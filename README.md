## Requirements
* A valid Microsoft Azure Subscription
* Python 3.9 installed
* Visual Studio Code installed

### VS Code Setup
To setup your VS Code go to the Extensions Tab and download + install the following extensions:
* Azure Account
* Azure App Service
* Azure Resources
* Azure Storage
